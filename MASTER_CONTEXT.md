# MASTER CONTEXT — Ichinotour Quotation

อัปเดตล่าสุด: 2026-07-19 (Asia/Bangkok)

## 1. เป้าหมายโครงการ

พัฒนาระบบสร้างและจัดการใบเสนอราคาของ Ichino Tour (Thailand) ให้ใช้งานจริงใน Browser รองรับการกรอกข้อมูล คำนวณราคา บันทึกหลายใบเสนอราคา จัดการลูกค้า ค้นหา สำรองข้อมูล และ Print/Save PDF

## 2. ขอบเขตถาวร

- ระบบใบเสนอราคา Ichinotour เท่านั้น
- ใช้ HTML แบบไฟล์เดียวเป็นฐานปัจจุบัน
- ข้อมูลเก็บใน Browser ด้วย `localStorage`
- ราคาใช้ JPY/Yen และแปลงเป็น THB ด้วยเรทที่ผู้ใช้กำหนด
- รองรับหมวด HOTEL, BUS, RESTAURANT, GUIDE / TOUR LEADER, TICKET / ENTRANCE และ OTHER
- ต้องรักษาความสามารถ Print/Save PDF
- ต้องแยกจาก JLOS และ SENEI KOTSU โดยเด็ดขาด

## 7. นโยบาย Checkpoint และความต่อเนื่อง

- ทุกครั้งที่งานหรือการเปลี่ยนแปลงสำเร็จ ต้องบันทึก checkpoint ลง GitHub ในรอบงานเดียวกัน
- `CURRENT_STATE.md` คือ checkpoint ล่าสุดสำหรับรับช่วงฉุกเฉิน
- ต้องอัปเดต `TASK_HISTORY.md`, `CHANGELOG.md` และ QA Report ที่เกี่ยวข้องตามผลจริง
- `MASTER_CONTEXT.md` อัปเดตเมื่อมีขอบเขตถาวร กฎถาวร สถาปัตยกรรม หรือการตัดสินใจหลักเปลี่ยนเท่านั้น
- ห้ามพึ่งข้อมูลสำคัญที่อยู่เฉพาะในบทสนทนา
